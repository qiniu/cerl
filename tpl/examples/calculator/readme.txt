+ Introduction

This application is a simple calculator. It supports:

++ REAL <op> REAL

Here <op> can be: operator+, operator-, operator*, operator/.
And here REAL can be any expression that return a real number.

++ <op> REAL

Here <op> can be unary operator+, unary operator-.

++ (REAL)

Yes, the calculator supports ().

++ sin(REAL), cos(REAL), pow(REAL, REAL), max(REAL, ...)

The calculator support sin, cos, pow, max functions.
Note that max function has variable arity.

+ Examples

Exp: 4 * -(3 + 2)
Result: -20

Exp: max(1, 2, pow(3,2), pow(2,3))
Result: 9
